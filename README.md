# 🎮 Retro Game Vault

## 📌 Description

Retro Game Vault is a web application that allows users to search and explore video games using real-time data from a public API. Users can filter and sort games based on different criteria and view game details.

---

## 🌐 API Used

RAWG Video Games Database API
https://rawg.io/apidocs
---

## ✨ Features

* Search games by name
* Filter by platform and genre
* Sort by rating and release date
* Add games to favorites (localStorage)
* Dark/Light mode toggle

---

## ⚙️ Technologies

* HTML
* CSS
* JavaScript
* Fetch API

---

## 🚀 How to Run

1. Clone the repository
2. Open `index.html` in browser
